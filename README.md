# Serendipity for Ghostty

Elegant, minimal, and clean color palette for your tools.

See other interfaces at the [official website](https://www.michaelandreuzza.com/vscode/serendipity/).

## Available themes

- **Midnight** — dark
- **Morning** — light
- **Sunset** — dark

## Installation

Add `config-file = ~/.config/ghostty/serendipity-midnight` to your Ghostty config.

Available files: `serendipity-midnight`, `serendipity-morning`, `serendipity-sunset`.
